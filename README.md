                             ..  𝗖𝗛𝗔𝗣𝗧𝗘𝗥  I  :    The “Introduction”.　　  

ᴡᴇʟᴄᴏᴍᴇ ᴛᴏ ᴍʏ ʀᴇᴀᴅᴍᴇ, ɴᴏᴡ ʟᴇᴛ ᴍᴇ ɪɴᴛʀᴏᴅᴜᴄᴇ ᴍʏꜱᴇʟꜰ. ʜᴇʟʟᴏ ɪ ᴀᴍ ᴋɪᴍᴀʟʟʏ ᴏʀ ʏᴏᴜ ᴄᴀɴ ᴄᴀʟʟ ᴍᴇ ꜰᴏʀ ꜱʜᴏʀᴛ ᴋɪᴍ ᴏʀ ᴘʏʜᴄᴏ.

ᴀɴʏᴡᴀʏꜱ ᴛʜᴀɴᴋꜱ ꜰᴏʀ ᴡᴀɴᴛᴇᴅ ᴛᴏ ʀᴇᴀᴅ ᴛʜɪꜱ, ᴀꜱ ɪ ꜱᴇᴇɴ ꜱᴏᴍᴇ ᴏꜰ ᴘᴇᴏᴘʟᴇ ᴊᴜꜱᴛ ʀᴇᴀᴅ ᴏɴʟʏ ꜱᴏᴍᴇ ᴘᴀʀᴛ ᴡɪᴛʜᴏᴜᴛ ʀᴇᴀᴅɪɴɢ ᴛʜɪꜱ ɪɴ ꜰᴜʟʟ ᴜɴᴅᴇʀꜱᴛᴀɴᴅɪɴɢ.ꜱᴏ ᴘʟᴇᴀꜱᴇ, ɪꜰ ʏᴏᴜ ʜᴀᴠᴇ ꜱᴏᴍᴇ ᴡᴏʀᴅꜱ ʏᴏᴜ ᴍᴀʏ ᴄᴀɴɴᴏᴛ ᴜɴᴅᴇʀꜱᴛᴏᴏᴅ, ʏᴏᴜ ᴄᴀɴ ᴀꜱᴋ ɪɴ ᴍʏ ᴀᴛᴀʙᴏᴏᴋ ᴏʀ, ɪꜰ ʏᴏᴜ ᴜɴᴅᴇʀꜱᴛᴀɴᴅ ᴀɴᴅ ꜱᴇᴇ ꜱᴏᴍᴇᴏɴᴇ ᴅᴏɴ'ᴛ ɪɴ ᴍʏ ᴀᴛᴀʙᴏᴏᴋ, ᴀɴᴅ ɢᴏ ᴀʜᴇᴀᴅ ɪꜰ ʏᴏᴜ ᴡᴀɴᴛ ʏᴏᴜ ᴄᴀɴ ᴀɴᴡᴇʀꜱ ɪᴛ ᴏʀ ᴄᴏʀʀᴇᴄᴛ ꜱᴏᴍᴇᴏɴᴇ.

ɴᴏᴡ, ᴀᴛʟᴇᴀꜱᴛ ʏᴏᴜ ᴋɴᴏᴡ ᴡʜᴏ ɪ ᴀᴍ, ʏᴏᴜ ᴍᴀʏ ᴘʀᴏɢʀᴀꜱꜱ ᴛᴏ ʀᴇᴀᴅ ꜰᴏʀ ᴍᴏʀᴇ ɪɴꜰᴏʀᴍᴀᴛɪᴏɴ ᴀʙᴏᴜᴛ ᴍᴇ ɪꜰ ʏᴏᴜ ʟɪᴋᴇ ᴛᴏ ᴜɴᴅᴇʀꜱᴛᴀɴᴅ, ᴛʜᴀᴛ ᴀʟʟ ꜰᴏʀ ɴᴏᴡ ɪʟʟ ꜱᴀʏ ɪɴ ᴛʜɪꜱ ɪɴᴛʀᴏᴅᴜᴄᴛɪᴏɴ ᴛʜᴀɴᴋꜱ ʏᴏᴜ. ʏᴏᴜ ᴍᴀʏ ᴘʀᴏᴄᴇꜱꜱ ᴀɴᴅ ꜱᴏʀʀʏ ꜰᴏʀ ᴀɴʏ ᴏꜰ ᴍʏ ᴇɴɢʟɪꜱʜ ᴛʜᴀᴛ ꜱᴏᴜɴᴅ ᴡʀᴏɴɢ ᴏʀ ʜᴏᴡ ɪᴛ ᴘᴜᴛ ᴀꜱ ᴇɴɢʟɪꜱʜ ɪꜱ ɴᴏᴛ ᴍʏ ꜰɪʀꜱᴛ ᴇᴠᴇʀ ʟᴀɴɢᴜᴀɢᴇ.
